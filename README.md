
This is a simple **AI Chatbot with Voice Support** created using **Python** and **Tkinter GUI**.  
It can answer questions, tell jokes, give the current time & date, and even speak its responses!  
This project was built by me as part of learning Python and AI basics.

---

## ✨ Features
- **GUI Interface** built using Tkinter.
- **Voice Support** – The chatbot speaks out its replies using `pyttsx3`.
- **Clear Chat Button** – Clear the chat history instantly.
- **Save Chat History** – All conversations are saved in a `chat_history.txt` file.
- **Custom Responses** (like favorite anime & games).
- **Fun Jokes** and real-time **date & time** responses.
to run it type in your cmd -python chatbot.py
